Chapter 1: The 50-Point Formal Grammar

Nova is strictly defined by an enterprise-grade EBNF grammar to ensure zero memory leaks.

program        = { statement } ;
statement      = declaration | assignment | expr_stmt | if_stmt | switch_stmt
               | for_stmt | while_stmt | repeat_stmt | foreach_stmt
               | function_decl | class_decl | interface_decl | enum_decl
               | struct_decl | module_decl | import_stmt | export_stmt ;

declaration    = [ "const" ] identifier "=" expression ;
assignment     = lvalue "=" expression ;
if_stmt        = "if" expression block [ "else" block ] ;
switch_stmt    = "switch" "(" expression ")" "{" { case_clause } "}" ;
